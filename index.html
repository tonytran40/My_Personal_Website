About this project

This project was inspired by the Pomodoro technique I used in college to stay focused and productive while studying. I built this timer as a way to practice my HTML, CSS, and JavaScript skills while creating something useful.

----------------------------------------------------------------

The application allows users to switch between work and break sessions, start/pause/reset the timer, and track progress with a circular progress indicator.

----------------------------------------------------------------

I also added features such as a task list, a Pomodoro session counter, a caffeine counter, and dark mode for a more comfortable viewing experience.

----------------------------------------------------------------

This project helped me practice DOM manipulation, event handling, and building interactive user interfaces in the browser.

What I learned and applied

  • Building page structure with HTML
  • Styling layouts with CSS
  • Using JavaScript variables, functions, and arrays
  • Manipulating the DOM to update the interface dynamically
  • Handling user interaction with event listeners
  • Using timers with setInterval for countdown functionality
